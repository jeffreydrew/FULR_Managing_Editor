FULR Managing Editor Slack Bot Documentation

Welcome to the comprehensive documentation for the FULR Managing Editor Slack Bot. This bot assists editors and writers with various tasks directly within Slack.

Introduction

The FULR Managing Editor Slack Bot is designed to streamline the workflow for editors' tasks. From assigning editors to writers to summarizing proposals, this bot leverages Google Sheets and GPT4 to automate and facilitate various editorial tasks.

A note on use: names must be typed exactly as in the form, and use of semi-colons must be observed when passing multiple arguments into a command.

How it Works

This bot is a serverless, Slack-integrated, AI-backed assistant which uses custom API calls to and from Google sheets so we can localize our editing workflow. It's able to access and manage all information on the reponse Google Sheet, as well as generate summaries and timelines for writers' proposals.

It operates primarily through slash commands, which are listed below, and returns responses to the user privately. Generative AI commands, however, are published to the channel for the benefit of all editors. When a command is invoked, for example, /brief-me [writer], the bot will look through all recorded responses for the latest proposal from the specified writer and generate a brief summary for that proposal.

Currently, this bot is in it's beta phase, and is only available to the FULR editing team. Future goals include expanding the bot's capabilities to include more AI-driven features, such as generating legal briefs and case law summaries, and integrating with other channels or even other law reviews.

Workflow Commands

/add-editor [writer: editor]

Assign an editor to a writer.

/writers-for [editor]

List all writers assigned to a specific editor.

/editor-for [writer]

Get the editor assigned to a specific writer.

/add-completion-date [name: date]

Add a completion date for a writer.

/meet [name: date]

Send a meeting request to a user.

/need-checkin

List users needing check-in.

/help

List all available commands.

Generative AI Commands

These commands only require the name of a writer, and will automatically find their latest proposal and process it with GPT4.

/brief-me [writer]

Get a brief summary of the latest proposal for a writer.

/guide [writer]

Create a timeline for a paper.

/caselaw [writer]

Provide links to legal resources.
