Documentation Last updated: Jun 29, 2024

The FULR Managing Editor Slack Bot is designed to streamline the workflow for editors' tasks. From assigning editors to writers to summarizing proposals, this bot leverages Google Sheets and GPT4 to automate and facilitate various editorial tasks.

A note on use: names must be typed exactly as in the form, and use of semi-colons must be observed when passing multiple arguments into a command.

How it Works

The FULR Managing Editor Slack Bot operates through a series of slash commands that interact with Google Sheets to manage and automate editorial tasks. Each command is designed to handle specific workflows, ensuring efficiency and accuracy in managing the editorial process. For more information on the technical details of the bot, see the Technical Details section.

Workflow Commands

/add-editor Copy

Assigns an editor to a writer.

Usage: /add-editor [writer (full name): editor (full name)]

Note

The response message will be sent to the entire channel. The individual editor will also be privately notified.

/writers-for Copy

Lists all writers assigned to a specific editor.

Usage: /writers-for [editor (full name)]

/editor-for Copy

Displays the editor assigned to a specific writer.

Usage: /editor-for [writer (full name)]

/add-completion-date Copy

Adds a completion date for a writer's task.

Usage: /add-completion-date [writer (full name): date]

/meet Copy

Schedules a meeting between an editor and a writer.

Usage: /meet [writer (full name): date]

/need-checkin Copy

Lists writers who need a check-in.

Usage: /need-checkin

/update-status Copy

Updates the status of a writer's task.

Usage: /update-status [writer (full name): status]

Note

The writer and updated status will be sent to the entire channel.

/get-status Copy

Retrieves the current status of a writer's task.

Usage: /get-status [writer (full name)]

/list-writers Copy

Lists all writers currently assigned to editors.

Usage: /list-writers

/help Copy

Provides help and usage information for all commands.

Usage: /help

Generative AI Commands

Note

All generative AI command responses will be sent to the entire channel.

/brief-me Copy

Generates a brief summary of a proposal.

Usage: /brief-me [writer (full name)]

/guide Copy

Provides a step-by-step guide based on input.

Usage: /guide [writer (full name)]

/caselaw Copy

Extracts keywords and phrases from a writer's proposal and finds matching cases through cluster searching with the Harvard Law School Caselaw Access Project and Courtlistener API.

Usage: /caselaw [writer (full name)]
