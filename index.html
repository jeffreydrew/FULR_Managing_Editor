Commands

  • /add-editor name: editor - Assign an editor to a writer.
  • /writers-for editor - List all writers assigned to a specific editor.
  • /meet username date - Send a meeting request to a user.
  • /brief-me writer - Get a brief summary of the latest proposal for a writer.
  • /get-editor writer - Get the editor assigned to a specific writer.
  • /help - List all available commands.

Usage

To use the commands, simply type the command in your Slack chat followed by the required parameters.

Examples

Here are some examples of how to use the commands:

  • /add-editor John Doe: Jane Smith
  • /writers-for Jane Smith
  • /meet @johndoe 2023-12-25
  • /brief-me John Doe
  • /get-editor John Doe
